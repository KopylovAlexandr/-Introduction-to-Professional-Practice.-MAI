Сборник лабораторных работ по программированию на языке **C**. Проекты охватывают ключевые темы: от алгоритмов сортировки и численных методов до низкоуровневой работы с файловыми потоками (текстовыми и бинарными), структурами данных и валидации пользовательского ввода.

---

## 📂 Обзор проектов и структура репозитория

### 🚉 1. Система управления расписанием поездов (`main.c`, `app_del_ed.c`)
Полноценное консольное CRUD-приложение, работающее с базой данных в виде бинарного файла (`.bin`). Демонстрирует работу со структурами, побайтовым чтением/записью и кастомными алгоритмами сортировки.
* **Функционал:** Добавление записей, удаление по номеру поезда, редактирование "на месте" (`fseek`), поиск по точным значениям и диапазонам (для времени).
* **Алгоритмическая база:** Реализованы кастомные сортировки методом пузырька (Bubble Sort) по возрастанию и убыванию для типов `int`, `double` и `string` (`strcmp`).
* **Компиляция и запуск:**

```bash
  gcc -Wall -Werror -Wextra main.c -o train_manager
  ./train_manager
```

### 🔤 2. Парсер и анализатор текста (`Lab1.c`)

Программа для обработки и валидации текстовых строк. Направлена на работу с указателями и динамическим токенизатором.

* **Логика работы:** Принимает строку слов, разделенных запятыми, заканчивающуюся точкой. Проверяет корректность формата, разбивает строку на токены с помощью `strtok`, проверяет ограничения на длину слова (до 10 символов) и количество слов (до 30). Выводит только уникальные слова.
* **Компиляция и запуск:**
```bash
gcc -Wall -Werror -Wextra Lab1.c -o text_parser
./text_parser
```

### 💾 3. Модификация бинарных и текстовых файлов (`Lab2(1).c`, `Lab2.c`)

Две вариации одной задачи, демонстрирующие разницу в обработке потоков данных на физическом уровне: через текстовый формат (`fprintf`/`fscanf`) и чистый бинарный (`fwrite`/`fread`).

* **Алгоритм:** Программа считывает массив вещественных чисел (до 100 элементов), подсчитывает количество положительных чисел, находит глобальный максимум и дописывает этот максимум в конец файла столько раз, сколько в файле было положительных чисел.
* **Компиляция и запуск:**
```bash
gcc -Wall -Werror -Wextra Lab2.c -o file_processor_txt
gcc -Wall -Werror -Wextra Lab2(1).c -o file_processor_bin

```



### 📐 4. Вычисление степенного ряда Тейлора (`Lab3.c`)

Реализация численных методов для аппроксимации математических функций.

* **Логика работы:** Вычисляет разложение функции $(1+x)^\alpha$ в степенной ряд. Поддерживает два режима работы:
1. Вычисление с заданной точностью (количеством знаков после запятой с математическим округлением `round`).
2. Вычисление для фиксированного количества членов ряда ($N$).


* **Компиляция и запуск:**
```bash
gcc -Wall -Werror -Wextra Lab3.c -o taylor_series -lm
./taylor_series

```

## 🛠️ Спецификация и требования к окружению

* **Компилятор:** GCC 9+ или Clang.
* **Стандарт языка:** C11 / C99.
* **Сборка:** Все файлы являются независимыми исполняемыми модулями. Для проверки компиляции рекомендуется использовать строгие флаги `-Wall -Werror -Wextra`, гарантирующие отсутствие утечек типов и потенциальных багов.

## 🧠 Ключевые навыки, продемонстрированные в коде

* Управление указателями, работа со строками и стандартными буферами ввода/вывода (`stdin`).
* Позиционирование указателя в файловых потоках (`fseek`, `ftell`, `rewind`).
* Применение базовых алгоритмов сортировки и поиска данных без использования сторонних библиотек.
* Реализация математических вычислений высокой точности.
